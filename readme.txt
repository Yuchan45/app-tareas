Descripcion:
Lee un archivo .json y enlista las tareas detalladas dentro de este. Para mas detalle leer el enunciado.

Ejecucion:
1- Abrir la terminal en la carpeta app-tareas.
2- Escribir comandos:
	a. node app.js listar (Enlista todas las tareas).
	b. node app.js crear "Nueva tarea". (Ej, node app.js crear Hacer ejercicio).
	c. node app.js filtrar "estado". (Ej, node app.js filtrar en progreso).
